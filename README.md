# AI Resume Builder

A modern, responsive AI Resume Builder that helps users create professional and ATS-friendly resumes with a live preview. The application allows users to enter their information, instantly preview the resume, upload a profile photo, download the resume as a PDF, print it, and save data automatically using Local Storage.

## Features

- Live Resume Preview
- ATS-Friendly Resume Design
- Profile Photo Upload
- Professional Resume Layout
- Skills Badges
- Projects Section
- Certifications Section
- Achievements Section
- Education Details
- Experience Section
- Languages Section
- GitHub, LinkedIn & Portfolio Links
- Print Resume
- Download Resume as PDF
- Dark Mode / Light Mode
- Auto Save using Local Storage
- Clear Form Functionality
- Responsive Design
- Keyboard Shortcut Support (Ctrl + P)

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Font Awesome
- HTML2PDF.js
- Local Storage API
